Tui mới học git nè :D

Các lệnh git cơ bản nên biết

  • git status [Dùng để kiểm tra trạng thái trong thư mục đang làm việc và staging area]
  • git diff id_commit_1 id_commit_2 [Dùng để so sánh sự khác nhau giữa các commit]
  • git reset file_name [Dùng để remove file ra khỏi staging area]
  • git fetch [Cập nhật sự thay đổi của remote repo xuống local repo]
  • git pull [kéo code từ remote repo xuống local repo]
  • git branch [Hiển thị các branch]
  • git checkout -b newbranch [Tạo ra thêm branch mới]
  • Đã xử lý conflict
